# Время и дата


##**$.unixtime**()

Возвращает текущее время в формате unixtime GMT0. Возможны арифметические операции над аргументами.

Примеры:

    // Возвращает текущее время в формате unixtime

        $.unixtime()

    // Возвращает время в формате unixtime текущее + 60 минут

        $.unixtime(%y-%m-%d %h:%i+60:%s)

    // Возвращает время в формате unixtime на текущую дату с установленным временем 02:00:00

        $.unixtime(%y-%m-%d 02:00:00)

##**$.date**(%y-%m-%d %h:%i:%s)

Преобразовывает дату в формат 2015-02-04 16:19:33. Возможны арифметические операции над аргументами.

Пример:

    // Вернет текущую дату/время в формате yyyy-MM-dd HH:mm:SS

        $.date(%y-%m-%d %h:%i:%s)

    // добавит +1 день к возвращаемой дате в формате yyyy-MM-dd HH:mm:SS

        $.date(%y-%m-%d+1 %h:%i:%s)
